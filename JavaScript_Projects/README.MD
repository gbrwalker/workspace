# JavaScript Projects

Esta pasta reúne projetos práticos desenvolvidos com **JavaScript**, com foco no aprendizado progressivo da linguagem e na aplicação de conceitos fundamentais de desenvolvimento front-end.

Os projetos são organizados de forma independente, permitindo estudar cada conceito isoladamente e compreender seu funcionamento na prática.

---

## Conteúdo Abordado

Os projetos desta pasta exploram, entre outros, os seguintes tópicos:

- Manipulação do DOM
- Eventos e propagação
- Trabalhar com tempo e animações
- Uso de APIs do navegador
- Interações com teclado e mouse
- Controle de estado no front-end
- Integração entre HTML, CSS e JavaScript

Cada diretório contém um projeto isolado, normalmente composto por arquivos HTML, CSS e JavaScript.

---

## Estrutura dos Projetos

- Cada pasta representa um projeto independente
- Os arquivos seguem a estrutura básica de aplicações front-end
- Não há dependências externas complexas
- Os projetos podem ser executados localmente, abrindo o arquivo `index.html` no navegador

---

## Objetivo

O objetivo desta coleção é:
- Consolidar fundamentos de JavaScript
- Desenvolver raciocínio lógico aplicado ao front-end
- Explorar comportamentos e APIs do navegador
- Criar uma base sólida para projetos web mais complexos

---

## Observações

- Os projetos não utilizam frameworks ou bibliotecas avançadas, priorizando JavaScript puro.
- Alguns projetos podem ter foco mais educacional do que visual.
- Melhorias e refatorações podem ser aplicadas futuramente conforme evolução técnica.
